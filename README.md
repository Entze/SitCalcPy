SitCalcPy is a Python library for doing stuff in Situation Calculus

## Installation

Use the package manager [conda](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html) to install
SitCalcPy.

```bash
conda env create
```

to update

```bash
conda env update
```
